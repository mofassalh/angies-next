# Restaurant White-Label Template

## Quick Setup for New Restaurant

### 1. Clone this repo
```bash
git clone https://github.com/mofassalh/angies-next.git my-restaurant
cd my-restaurant
```

### 2. Run setup script
```bash
./setup.sh
```

### 3. Follow the prompts
- Restaurant name
- Tagline
- Primary color
- Supabase credentials

### 4. Deploy to Vercel
```bash
git push origin main
```
Then connect to Vercel for auto-deploy.

## What's included
- ✅ Customer ordering portal
- ✅ Location selection
- ✅ Menu with categories
- ✅ Cart & checkout (pickup + delivery)
- ✅ Loyalty points system
- ✅ Coupon codes
- ✅ Order tracking
- ✅ User profiles
- ✅ Featured items
- ✅ Promo sections

## Configuration
Edit `restaurant.config.js` to customize:
- Restaurant name & branding
- Colors
- Locations
- Features (enable/disable loyalty, delivery, etc.)

## Admin Panel
Separate repo: `angies-admin`
Same setup process applies.
